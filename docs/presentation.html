Key Takeaways

  • Vision: Provide a realistic EMR simulation to help nursing students practice safely.
  • Recent Work: Improved Pediatric Assessment form and updated dev environments to run locally.
  • Testing: Expanded automated tests for authentication, patient creation, and patient lists.
  • Features: Admin features such as sharing patients and adding individual students were added.
  • Next Steps: Continue refining assessments, add additional tests, incorporate feedback from nursing faculty, and prepare the codebase for future teams.
