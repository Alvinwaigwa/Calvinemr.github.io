Key Takeaways

  • What is an EMR: Electronic Medical Record system for healthcare documentation.
  • Benefits: Training nursing students with EMR allows safe practice without risking actual patients.
  • Project Overview: Multi-year initiative building on previous teams' work, adding features and comprehensive tests.
  • Additions Made: Pediatric and OB/GYN assessments, automatic PDF generation, improved usability.
  • Testing: Established automated testing framework using Playwright for authentication, patient management, and assessments.
  • Design Norms: Worked with nursing professors to ensure appropriateness, transparency, and caring in the design.
  • Demo: Available via the embedded presentation above and YouTube preview.
  • Future Plans: Continue refining assessments, expanding tests, incorporating feedback, and preparing for future teams.
